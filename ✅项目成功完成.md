### 🚀 启动项目

1. **访问应用** - 打开 `http://localhost:3000`
2. **测试连接** - 点击"测试后端连接"按钮
3. **配置 API** - 点击"API 配置"设置 Nano Banana API 密钥
4. **配置数据库** - 点击"数据库"设置 MySQL 连接信息
5. **开始使用** - 配置完成后即可使用所有功能
